
- **assets/**: Contains the source files.
- **generated/**: Contains the generated files.

## Prerequisites

- Node.js (>= 18)
- pnpm (>= 7)

## Installation

1. Clone the repository:
    ```sh
    git clone <repository-url>
    cd assets
    ```

2. Install dependencies:
    ```sh
    pnpm install
    ```

## Usage

To run the conversion

```sh
pnpm run build
